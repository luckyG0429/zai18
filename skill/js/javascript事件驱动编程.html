事件驱动编程

javascript 是一门事件驱动编程语言，如果为特定事件注册了一段代码，这段代码将会在事件被触发时执行。

实例化一个promise对象。new const promise = new Promise((resolve,reject)=>{ if(/*异步操作成功*/){ resolve(value); }else{ reject(error); } })

( 使用promise.prototype.then(onFulfilled,onRejected)方法来为promise对象指定已解决和被拒绝的情况下触发的回调函数。 const p2= new Promise((resolve,reject)=>{ resovle('成功'); //reject('失败'); }) p2.then(value=>console.log(value),reason=>console.log(reason));

使用promise.prototype.catch(onRecjected)只处理Promise对象被拒绝的情况 该方法的行为和调用和promise.prototype.then(undefined,onRejected)相同 const p1 = new Promise((resolve,reject)=>{ resolve('成功') }) p1.then(value=>{ console.log(value); throw '哦 ， nono'; }).catch(value=>console.log(value));
